# Task 3 - Smart Home Lighting Automation System

## Project Overview

This project focuses on the design of an automated smart home lighting system using motion detection technology. The system automatically controls lighting based on room occupancy to improve convenience and reduce energy consumption.

## Objectives

* Design an automated lighting control system.
* Detect human movement using sensors.
* Automatically switch lights ON and OFF.
* Improve energy efficiency.

## Components Used

* PIR Motion Sensor
* Controller / Arduino
* Relay Module
* Light Bulb
* Power Supply
